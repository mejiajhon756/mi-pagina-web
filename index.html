Proyectos Destacados

Calculadora de Estadísticas Descriptivas

Objetivo: Crear una herramienta web interactiva para calcular medidas de tendencia central (media, mediana, moda) y de dispersión (varianza, desviación estándar) a partir de un conjunto de datos.

Mi Rol: Desarrollé el frontend con HTML, CSS y JavaScript para la captura de datos y la visualización de resultados, así como la lógica para realizar los cálculos estadísticos.

Resultados: La aplicación procesa datos en tiempo real, proporcionando una herramienta útil para estudiantes y analistas. El código fue optimizado para un rendimiento rápido en el navegador.

Dashboard de Visualización de Datos

Objetivo: Diseñar un dashboard dinámico para visualizar un conjunto de datos de ventas utilizando gráficos interactivos como barras, líneas y pasteles.

Mi Rol: Fui responsable de la integración de la librería Chart.js, la preparación de los datos (limpieza y formato) y el diseño de una interfaz intuitiva para el usuario.

Resultados: El dashboard permite una comprensión clara de los patrones y tendencias en los datos, facilitando la toma de decisiones. Fue presentado como proyecto final de la materia de '''Visualización de Información'''.
