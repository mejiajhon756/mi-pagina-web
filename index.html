Análisis Estadístico: Popularidad de Lenguajes de Programación

A continuación, se presenta un análisis descriptivo sobre la tendencia de popularidad de tres de los lenguajes de programación más influyentes: Python, JavaScript y Java, durante el período 2019-2023. Los datos se basan en una simulación de métricas de uso extraídas de repositorios de código, encuestas a desarrolladores y ofertas de empleo.

Análisis de Tendencia

Se observa una tendencia lineal positiva muy marcada para Python, con una tasa de crecimiento promedio anual del 15%. JavaScript muestra una tendencia estable pero con alta varianza, manteniéndose como el lenguaje dominante en el desarrollo web. Java, por otro lado, presenta una ligera tendencia negativa, con una disminución promedio del 3% anual, aunque sigue siendo un pilar en el sector corporativo.

Estadísticas Descriptivas Clave

La media de popularidad de JavaScript en el período fue la más alta. Sin embargo, la desviación estándar de Python fue la menor entre los que crecen, indicando un crecimiento más consistente y predecible. Se encontró una correlación negativa débil (-0.4) entre el crecimiento de Java y el de Python, sugiriendo que el auge de Python en áreas como Data Science e IA podría estar atrayendo a desarrolladores que antes se hubieran inclinado por Java.

Conclusión del Análisis

Python se perfila como el lenguaje con mayor proyección de crecimiento. JavaScript mantendrá su dominio en la web, y Java seguirá siendo relevante en sistemas legados y de gran escala, aunque con una base de usuarios que decrece lentamente. La elección de un lenguaje depende fuertemente del dominio de aplicación.
